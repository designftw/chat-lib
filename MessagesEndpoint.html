getMessagesForAlias(ownAlias, interlocutors, sinceTime) → {Promise.<Array.<Message>>}

Get all the messages sent or received by the alias and the passed in interlocutors. If no interlocutors are passed in, returns all the messages sent or received by the alias. If for example the passed in interlocutors contains the id for the alias Sally, and the passed in alias is Jack, then this method returns all messages sent by Jack to Sally, or by Sally to Jack. If there are multiple interlocutors this method returns all messages sent by Jack to any of the interlocutors, or by any of the interlocutors to Jack. Note this does not mean that the message has to be sent to all of the interlocutors to be here. Any pair suffices.
Parameters:
Name Type Description
ownAlias string an alias associated with the currently logged in account
interlocutors Array.<string> | undefined An optional list of aliases which are either the sender or receiver of the returned messages..
sinceTime Date | undefined An optional date which filters messages to those which were sent after this time.
